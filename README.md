Group project code for Mairin O'Shaughnessy, Katherine Yan, and Kaitlyn Zhang

Main notes from [Local Development Guide](https://docs.google.com/document/d/1ZJRd_iI4eL5dYLiOf87SIPuMNaDM_yJHCihtkyXwC6k/edit?tab=t.0):
1. Start up Docker Desktop
2. Press ⌘⇧P / Ctrl⇧P → “Dev Containers: Reopen in Container” (up to 15 min first time)
3. Run code with `.venv/bin/python src/code.py`
4. Update dependencies with `.venv/bin/pip install -U -r requirements.txt`
